# Dummy Project

A simple starter project to demonstrate active development.

## Overview
This is a lightweight project showcasing best practices and modern development patterns.

## Features
- Clean project structure
- Easy to understand
- Ready for expansion

## Getting Started

### Prerequisites
- Python 3.8 or higher

### Installation
```bash
git clone https://github.com/Ganeshmunishetty/Dummy.git
cd Dummy
pip install -r requirements.txt
```

### Usage
```bash
python main.py
```
